Reaction Diffusion Assignment <br/>
Sarah Hatch <br/>

I created a reaction diffusion simulation with the option to use a style map. I implemented a standard feed and kill rate gradient map which shows the range of usable feed and kill values. I then mapped these range of values to a radial effect, where the distance from the center effects both the feed and the kill values. The other controls I implemented where for the diffuse rates for chemicals A and B, the constant feed and kill values and a slider for scaling the radial effect. When building the UI I specifically focues on shrinking the ranges to only their useful values and allowing edits to only sliders that where relevent to the mode currently running.
